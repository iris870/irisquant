# IrisQuant v2.1

Multi-agent quantitative trading system dashboard with real-time log streaming and market monitoring.

## Features
- Real-time Binance Market Data
- PM2 Service Monitoring (13 Agents)
- Semantic Log Search & Highlighting
- FastAPI + WebSocket Backend
- Tailwind CSS + JetBrains Mono Frontend

## Structure
- `/web`: Dashboard frontend and backend.
- `/scripts`: Administrative and maintenance tools.
- `/data`: Knowledge DB and market snapshots (SQLite).
